XHTML

XHTML stands for eXtensible HyperText Markup Language. The ”x” came from XML or eXtensible Mark-up Language. Though different from HTML, XHTML also uses the filename extension .html. The content of the actual coding might change because of XHTML conventions but it still holds the same filename extension. It is stricter and cleaner version of HTML. It aimed to replace HTML. Also, all new Web browsers support XHTML.
